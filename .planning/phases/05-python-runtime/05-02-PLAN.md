    **validation.py** -- Core orchestration:
    ```python

    from pydantic import BaseModel, ValidationError
    ```

    `_is_primitive(return_type)` helper: returns True if return_type is str, int, float, bool, or a Literal type (check `get_origin(return_type) is Literal`).

    `_validate_primitive(raw: str, return_type: type) -> Any` helper:
    - `json.loads(raw)` to get the value
    - If return_type is str/int/float/bool: `isinstance(value, return_type)` check, raise ValueError if wrong type
    - If Literal: check `value in get_args(return_type)`, raise ValueError if not a valid literal value
    - Return the validated value

    `async def validate_or_retry(provider, messages, model_id, return_type, max_retries=3, **kwargs) -> Any`:
    - Loop `max_retries` times
    - Call `provider.send_prompt(messages, model_id, **kwargs)`
    - If `_is_primitive(return_type)`: call `_validate_primitive(raw, return_type)`, catch ValueError/json.JSONDecodeError
    - Else (BaseModel subclass): call `return_type.model_validate_json(raw)`, catch ValidationError
    - On any validation failure: fire ValidationFailureEvent, append error as user message, continue loop
    - On success: return result
    - After loop: raise EamlValidationError(model_id=model_id, attempts=max_retries, last_response=raw, errors=errors)
    - Any non-validation exception from provider: DO NOT catch, let it bubble up

    `async def execute_prompt(*, model, messages, return_type, temperature=None, max_tokens=None, max_retries=3) -> Any`:
    - Extract `provider_name = model["provider"]`, `model_id = model["model_id"]`
    - `provider = get_provider(provider_name)`
    - Fire CallStartEvent
    - `start = time.time()`
    - try: call validate_or_retry, fire CallEndEvent with duration, return result
    - except EamlError: re-raise as-is (includes EamlValidationError, EamlConfigError, EamlProviderError)
    - except Exception as exc: wrap in EamlProviderError(provider=provider_name, message=str(exc))
