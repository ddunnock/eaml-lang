5. `check_bad_model_exits_one_with_cap010`:
   - Run `eamlc check examples/06-capability-error/bad_model.eaml`

   - Assert `tmpdir/bad_model.py` does NOT exist (no output on error)

   - Run `eamlc check examples/06-capability-error/bad_model.eaml`

For mypy tests, if mypy is not installed, the test should print a skip message and return (not fail). Use this pattern:
```rust
fn mypy_available() -> bool {
```
