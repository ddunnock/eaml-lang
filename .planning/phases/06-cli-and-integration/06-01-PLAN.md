**Clap structs:**
```rust
use clap::{Parser, Subcommand};
use std::path::PathBuf;

#[derive(Parser)]
#[command(name = "eamlc", version, about = "The EAML compiler")]
struct Cli {
    #[command(subcommand)]
    command: Commands,
}

#[derive(Subcommand)]
enum Commands {
    /// Compile an EAML file to Python
    Compile {
        /// Input .eaml file
        file: PathBuf,
        /// Output directory (default: same as input file)
        #[arg(short, long)]
        output: Option<PathBuf>,
    },
    /// Check an EAML file for errors without generating output
    Check {
        /// Input .eaml file
        file: PathBuf,
    },
    /// Compile and run an EAML file
    Run {
        /// Input .eaml file
        file: PathBuf,
        /// Output directory (default: same as input file)
        #[arg(short, long)]
        output: Option<PathBuf>,
    },
}
```

**Exit codes** (constants, not enum -- simpler):
```rust
const EXIT_SUCCESS: i32 = 0;
const EXIT_COMPILE_ERROR: i32 = 1;
const EXIT_IO_ERROR: i32 = 2;
const EXIT_RUNTIME_ERROR: i32 = 3;
```

**main() function:**
- Parse CLI args with `Cli::parse()`
- Match on command, dispatch to `cmd_compile`, `cmd_check`, `cmd_run`
- Call `std::process::exit(code)` with the returned exit code

**read_source(path: &Path) -> Result<String, i32> helper:**
- Read the file with `std::fs::read_to_string`
- On error: print `error: could not read file '{path}': {err}` to stderr, return `Err(EXIT_IO_ERROR)`
- Validate file has `.eaml` extension; if not, print warning but continue

**run_pipeline(source: &str, filename: &str) -> (Option<String>, Vec<Diagnostic>, bool) helper:**
- Call `eaml_parser::parse(source)` to get `ParseOutput`
- Collect parse diagnostics into `all_diagnostics: Vec<Diagnostic>`
- Check if any diagnostic has `severity == Severity::Error || severity == Severity::Fatal`
- If parse errors exist: return `(None, all_diagnostics, true)`
- Call `eaml_semantic::analyze(&parse_output, source)` to get `AnalysisOutput`
- Extend `all_diagnostics` with `analysis.diagnostics`
- Check for errors again
- If semantic errors exist: return `(None, all_diagnostics, true)`
- Call `eaml_codegen::generate(&parse_output, &analysis, source, filename)` to get Python string
- Return `(Some(python_code), all_diagnostics, false)`

**render_and_summarize(filename: &str, source: &str, diagnostics: &[Diagnostic]) helper:**
- If diagnostics is empty, return immediately
- Create `SimpleFiles::new()`, add source with `files.add(filename, source)`
- Call `eaml_errors::render::render_diagnostics(&files, diagnostics)`
- Call `print_summary(diagnostics)`

**print_summary(diagnostics: &[Diagnostic]) helper:**
- Count errors (Error + Fatal severity) and warnings (Warning severity)
- If error_count > 0: `eprintln!("error: aborting due to {N} previous error(s){warning_suffix}")`
  - warning_suffix: `"; {M} warning(s) emitted"` if warning_count > 0, else empty
- Else if warning_count > 0: `eprintln!("warning: {M} warning(s) emitted")`

**cmd_compile(file: &Path, output_dir: Option<&Path>) -> i32:**
- Call `read_source(file)`, propagate error exit code
- Derive `filename` from file path (e.g., `file.file_name().unwrap().to_str().unwrap()`)
- Call `run_pipeline(&source, filename)`
- Call `render_and_summarize(filename, &source, &diagnostics)`
- If `has_errors`: return `EXIT_COMPILE_ERROR`
- Determine output path: if `output_dir` is Some, use `output_dir.join(stem + ".py")`; otherwise use `file.with_extension("py")`
  - Stem: `file.file_stem().unwrap().to_str().unwrap()`
- Write python_code to output path with `std::fs::write`
- On write error: print `error: could not write '{path}': {err}` to stderr, return `EXIT_IO_ERROR`
- Print to stderr: `Compiled {filename} -> {output_path.display()}`
- Return `EXIT_SUCCESS`

**cmd_check(file: &Path) -> i32:**
- Call `read_source(file)`, propagate error exit code
- Call `run_pipeline(&source, filename)` (same as compile, but ignore the generated code)
- Call `render_and_summarize(filename, &source, &diagnostics)`
- If `has_errors`: return `EXIT_COMPILE_ERROR`
- Print to stderr: `{filename}: no errors found`
- Return `EXIT_SUCCESS`

**cmd_run(file: &Path, output_dir: Option<&Path>) -> i32:**
- First, compile (same as cmd_compile logic). If compile fails, return that exit code.
- Discover Python interpreter: try `std::process::Command::new("python3").arg("--version").output()`. If that fails (not found), try `"python"`. If neither works, print `error: Python interpreter not found. Ensure python3 or python is on PATH.` to stderr and return `EXIT_RUNTIME_ERROR`.
- Run `Command::new(python_cmd).arg(&output_path).status()`
- If status is not success, return `EXIT_RUNTIME_ERROR`
- Return `EXIT_SUCCESS`
