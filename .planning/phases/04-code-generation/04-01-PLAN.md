Create `src/types.rs` with:

1. `pub fn emit_type_annotation(resolved: &ResolvedType, ast: &Ast, interner: &Interner) -> String`:
   Match on ResolvedType variants using the mapping from TYPESYSTEM.md 10.3:
   - `Primitive("string")` -> `"str"`
   - `Primitive("int")` -> `"int"`
   - `Primitive("float")` -> `"float"`
   - `Primitive("bool")` -> `"bool"`
   - `Primitive("null")` -> `"None"`
   - `Schema(id)` -> `interner.resolve(&ast[*id].name).to_string()`
   - `Array(inner)` -> `format!("List[{}]", emit_type_annotation(inner, ast, interner))`
   - `Optional(inner)` -> `format!("Optional[{}]", emit_type_annotation(inner, ast, interner))`
   - `LiteralUnion(members)` -> `format!("Literal[{}]", members.iter().map(|m| format!("\"{}\"", m)).collect::<Vec<_>>().join(", "))`
   - `Error` -> `"Any".to_string()`

2. `pub fn is_optional(resolved: &ResolvedType) -> bool`:
   Returns true if outermost is `ResolvedType::Optional(_)`.

3. `pub fn emit_field_line(field_name: &str, resolved: &ResolvedType, type_expr: &TypeExpr, ast: &Ast, interner: &Interner, source: &str) -> String`:
   Builds the complete field declaration line for a Pydantic model field:
   - Get type annotation via `emit_type_annotation`
   - Check if type_expr is `TypeExpr::Bounded { base, params, .. }`:
     - For base "float" or "int": positional params[0]=min params[1]=max -> `Field(ge={min}, le={max})`; named params: resolve name via interner, "min" -> "ge", "max" -> "le"
     - For base "string": "min"/"minLen" -> "min_length", "max"/"maxLen" -> "max_length"
     - Extract values from source via `source[param.value_span.clone()]`
     - Return `"{name}: {annotation} = Field({kwargs})"`
   - If is_optional(resolved) and NOT bounded: return `"{name}: {annotation} = None"`
   - Otherwise: return `"{name}: {annotation}"`

4. `pub struct ImportTracker` with methods:
   - `new() -> Self` with fields: `pydantic: HashSet<&'static str>`, `typing: HashSet<&'static str>`, `eaml_runtime: HashSet<&'static str>`
   - `need_base_model(&mut self)` -> inserts "BaseModel" into pydantic set
   - `need_field(&mut self)` -> inserts "Field" into pydantic set
   - `need_optional(&mut self)` -> inserts "Optional" into typing set
   - `need_list(&mut self)` -> inserts "List" into typing set
   - `need_literal(&mut self)` -> inserts "Literal" into typing set
   - `need_execute_prompt(&mut self)` -> inserts "execute_prompt" into eaml_runtime set
   - `need_agent(&mut self)` -> inserts "Agent" into eaml_runtime set
   - `need_tool_metadata(&mut self)` -> inserts "ToolMetadata" into eaml_runtime set
   - `track_type(resolved: &ResolvedType)` -> recursively walks resolved type calling need_optional/need_list/need_literal as appropriate
   - `emit_imports(&self, writer: &mut CodeWriter)` -> emits sorted `from X import Y` lines for non-empty sets

Tests in `tests/types.rs`:
