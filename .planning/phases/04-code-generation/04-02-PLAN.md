  <action>
1. Create `src/emitters.rs`. Add `pub mod emitters;` to lib.rs.

2. Implement `pub fn emit_schema(schema: &SchemaDecl, ast: &Ast, interner: &Interner, type_annotations: &TypeAnnotations, source: &str, writer: &mut CodeWriter, imports: &mut ImportTracker)`:
   - Schema name stays PascalCase per CONTEXT.md locked decision
   - `imports.need_base_model()`
   - `writer.writeln(&format!("class {}(BaseModel):", interner.resolve(&schema.name)))`
   - `writer.indent()`
   - For each field in schema.fields:
     - Get resolved type from `type_annotations.type_exprs[&field.type_expr]`
     - Track imports via `imports.track_type(&resolved)`
     - Get field name via `interner.resolve(&field.name)`
     - Get type_expr from `&ast[field.type_expr]`
     - If bounded type expr, call `imports.need_field()`
     - Use `emit_field_line(field_name, &resolved, type_expr, ast, interner, source)` to get the line
     - `writer.writeln(&line)`
   - If no fields, `writer.writeln("pass")`
   - `writer.dedent()`

3. Implement helper `fn emit_expr_value(expr_id: ExprId, ast: &Ast, interner: &Interner, source: &str) -> String`:
   - Match on Expr variants:
     - `IntLit(span)` -> `source[span.clone()].to_string()`
     - `FloatLit(span)` -> `source[span.clone()].to_string()`
     - `StringLit(ts)` -> emit as Python string (extract text, wrap in quotes)
     - `BoolLit(true, _)` -> `"True"`
     - `BoolLit(false, _)` -> `"False"`
     - `NullLit(_)` -> `"None"`
     - `Ident(spur, _)` -> `interner.resolve(spur).to_string()`
     - `TemplateStr(ts)` -> emit as f-string
     - Other -> `"None"` fallback

4. Implement `pub fn emit_let(decl: &LetDecl, ast: &Ast, interner: &Interner, type_annotations: &TypeAnnotations, source: &str, writer: &mut CodeWriter, imports: &mut ImportTracker)`:
   - Get name: `interner.resolve(&decl.name)`
   - Get resolved type from type_annotations
   - Get type annotation via `emit_type_annotation`
   - Get value via `emit_expr_value(decl.value, ...)`
   - `writer.writeln(&format!("{}: {} = {}", name, annotation, value))`

5. Write snapshot tests in `tests/schemas.rs`:
   - Test minimal schema (string + int fields)
   - Test bounded float field
   - Test optional field with = None
   - Test literal union field
   - Test array field
   - Test all-type-variants schema
   Use `parse_and_analyze()` from test_helpers, then create a CodeWriter + ImportTracker, call emit_schema, and snapshot the output.

   Implement helper `fn extract_template_text(ts: &TemplateString, source: &str) -> String`:
   - Collect all TemplatePart::Text spans, concatenate: `source[span.clone()]` for each
   - This extracts the plain string value from template strings (model id and provider are always plain text, no interpolations)

2. Implement `pub fn emit_model(model: &ModelDecl, ast: &Ast, interner: &Interner, source: &str, writer: &mut CodeWriter)`:
   - Model name converts to UPPER_SNAKE_CASE + "_CONFIG" suffix per CONTEXT.md locked decision
   - `let config_name = to_config_name(interner.resolve(&model.name));`
   - Extract provider text: `extract_template_text(&model.provider, source)`
   - Extract model_id text: `extract_template_text(&model.model_id, source)`
   - Extract capabilities: `model.caps.iter().map(|(spur, _)| interner.resolve(spur).to_string()).collect::<Vec<_>>()`
   - Emit:
     ```
     {CONFIG_NAME} = {
         "provider": "{provider}",
         "model_id": "{model_id}",
         "capabilities": [{caps_list}],
     }
     ```
   - For capabilities list: if empty, emit `[]`; otherwise format as `["cap1", "cap2"]`
   - Use writer.writeln for each line, writer.indent/dedent for the dict body
