**`parse_program(&mut self) -> ParseOutput`** -- Grammar production [24]:
```rust
pub fn parse_program(&mut self) -> ParseOutput {
    let start = 0usize;
    let mut declarations = Vec::new();

    while !self.at(TokenKind::Eof) {
        if self.error_limit_reached() {
            break;
        }

        let decl = match self.peek() {
            TokenKind::KwImport => self.parse_import_decl(),
            TokenKind::KwModel => self.parse_model_decl(),
            TokenKind::KwSchema => self.parse_schema_decl(),
            TokenKind::KwPrompt => self.parse_prompt_decl(),
            TokenKind::KwTool => self.parse_tool_decl(),
            TokenKind::KwAgent => self.parse_agent_decl(),
            TokenKind::KwLet => self.parse_let_decl(),
            // Post-MVP reserved keywords
            TokenKind::KwPipeline => self.parse_reserved_decl(ErrorCode::Syn080, "Pipeline declarations are not supported in EAML v0.1."),
            TokenKind::KwEnum => self.parse_reserved_decl(ErrorCode::Syn082, "Enum declarations are not supported in EAML v0.1."),
            // @ annotation
            TokenKind::At => {
                let span = self.peek_span();
                self.emit_error(ErrorCode::Syn090, "@annotations are not supported in EAML v0.1.".into(), span.clone(), "reserved syntax".into());
                self.advance();
                // Skip to next declaration
                self.synchronize();
                DeclId::Error(span)
            }
            _ => {
                let span = self.peek_span();
                self.emit_error(ErrorCode::Syn060, format!("expected declaration, found {:?}", self.peek()), span.clone(), "expected model, schema, prompt, tool, agent, import, or let".into());
                self.synchronize();
                DeclId::Error(span)
            }
        };

        declarations.push(decl);
    }

    let end = if self.pos > 0 { self.previous_span().end } else { 0 };
    let program = Program { declarations, span: start..end };

    ParseOutput {
        ast: std::mem::take(&mut self.ast),
        program,
        diagnostics: std::mem::take(&mut self.diagnostics),
        interner: std::mem::take(&mut self.interner),
    }
}
```

Note: `std::mem::take` requires `Ast`, `Interner` etc. to implement `Default`. Make sure `Ast::new()` is used as `Default::default()`. Add `impl Default for Ast` and for `Interner` (already has it).

**`parse_import_decl(&mut self) -> DeclId`** -- Grammar production [26]:
1. expect KwImport, record start span
2. LL(2) disambiguation: if peek is KwPython -> Python import, else EAML import
3. EAML: parse template string (the path), optionally check for contextual "as" + expect_ident for alias
4. Python: expect KwPython, parse template string (module), optionally "as" + ident
5. eat(Semicolon)
6. Alloc import, return DeclId::Import(id)
7. On parse error: emit diagnostic, synchronize, return DeclId::Error(span)

**`parse_model_decl(&mut self) -> DeclId`** -- Grammar production [27]:
1. expect KwModel, expect_ident (name), expect Eq
2. Expect contextual "Model", expect LParen
3. Fixed order: expect_contextual("id"), Colon, parse_template_string, Comma
4. expect_contextual("provider"), Colon, parse_template_string, Comma
5. expect_contextual("caps"), Colon, LBracket, parse cap list (comma-separated idents, allow empty), RBracket
6. expect RParen, eat(Semicolon)
7. Alloc model, return DeclId::Model(id)
8. On any expect failure: emit diagnostic, synchronize, return DeclId::Error

**`parse_schema_decl(&mut self) -> DeclId`** -- Grammar production [29]:
1. expect KwSchema, expect_ident (name)
2. Check for Post-MVP `extends`: if at(KwExtends), emit SYN083 "Schema inheritance is not supported in EAML v0.1.", synchronize, return Error
3. expect LBrace
4. Loop parsing field definitions until RBrace or Eof:
   - Each field: expect_ident (field name), Colon, parse_type_expr, eat(Comma) [optional trailing comma per grammar]
   - If next token is not Ident and not RBrace, emit error and break
5. expect RBrace, eat(Semicolon)
6. Alloc schema, return DeclId::Schema(id)

**`parse_let_decl(&mut self) -> DeclId`** -- Grammar production [41]:

2. Alloc let, return DeclId::Let(id)

**`parse_tool_body(&mut self) -> ToolBody`** -- Grammar productions [35]-[37]:
1. expect LBrace
2. Left-factored dispatch per grammar:
   - If at KwPythonBridge: this is `python %{` -- advance (consumes KwPythonBridge token), then expect PythonBlock token (the opaque content), record code_span. The closing `}%` is handled by the lexer. Then expect RBrace (the tool body closing `}`).
   - Wait -- the lexer emits KwPythonBridge + PythonBlock tokens for `python %{ ... }%`. So after LBrace:
     - If peek is KwPythonBridge: advance, expect PythonBlock (record span), expect RBrace -> ToolBody::PythonBridge { description: None, code_span, span }
     - If at_contextual("description"): advance, expect Colon, parse_template_string (description), then expect KwPythonBridge, PythonBlock, expect RBrace -> ToolBody::PythonBridge { description: Some(desc), code_span, span }
     - If at RBrace: empty body -> ToolBody::Empty(span) (semantic analysis will emit SEM040)
     - Else: this is a native body (Post-MVP) -> emit Syn050 "Native tool bodies are not supported in EAML v0.1. Use python %{ }% for tool implementations.", skip to matching RBrace -> ToolBody::Native { stmts: vec![], span }
3. expect RBrace (if not already consumed)

**`parse_agent_decl(&mut self) -> DeclId`** -- Grammar productions [38]-[40]:
1. expect KwAgent, expect_ident (name)
2. expect LBrace
3. Loop until RBrace or Eof, matching contextual keywords:
   - "model": Colon, expect_ident -> AgentField::Model
   - "tools": Colon, LBracket, comma-separated idents, RBracket -> AgentField::Tools
   - "system": Colon, parse_template_string -> AgentField::System
   - "max_turns": Colon, expect IntLit -> AgentField::MaxTurns(span)
   - "on_error": Colon, parse_error_policy -> AgentField::OnError
   - Other: emit Syn061 "unexpected field in agent body"
4. expect RBrace, eat(Semicolon)
5. Alloc agent, return DeclId::Agent(id)

**`parse_error_policy(&mut self) -> ErrorPolicy`** -- Grammar production [40]:

- If at_contextual("retry"): advance, expect LParen, record IntLit span, expect RParen, expect_contextual("then"), expect_contextual("fail"), return ErrorPolicy::RetryThenFail { retries_span }
- Else: emit error, return ErrorPolicy::Fail as default
