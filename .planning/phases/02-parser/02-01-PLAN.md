Create `crates/eaml-parser/src/parser.rs`:

```rust
use eaml_errors::{Diagnostic, ErrorCode, Severity, Span};
use eaml_lexer::{Interner, Token, TokenKind};
use lasso::Spur;
use crate::ast::*;

pub struct Parser {
    tokens: Vec<Token>,
    pos: usize,
    source: String,
    pub(crate) ast: Ast,
    pub(crate) diagnostics: Vec<Diagnostic>,
    pub(crate) interner: Interner,
}
```

**Constructor:** `Parser::new(source: String, tokens: Vec<Token>, interner: Interner, lex_diagnostics: Vec<Diagnostic>) -> Self` -- stores source, tokens, interner, seed diagnostics from lexer, pos=0, ast=Ast::new().

**Cursor methods:**
- `peek(&self) -> TokenKind` -- returns `self.tokens[self.pos].kind` (clone/copy since TokenKind: Copy). If pos >= tokens.len(), return TokenKind::Eof.
- `peek_span(&self) -> Span` -- returns `self.tokens[self.pos].span.clone()`
- `peek_at(&self, offset: usize) -> TokenKind` -- lookahead by offset (for LL(2) points)
- `advance(&mut self) -> &Token` -- returns `&self.tokens[self.pos]` and increments pos (clamped to tokens.len())
- `previous_span(&self) -> Span` -- returns `self.tokens[self.pos.saturating_sub(1)].span.clone()`
- `at(&self, kind: TokenKind) -> bool` -- uses `std::mem::discriminant` comparison to handle Ident(Spur) payload. `std::mem::discriminant(&self.peek()) == std::mem::discriminant(&kind)`
- `at_ident(&self) -> bool` -- `matches!(self.peek(), TokenKind::Ident(_))`
- `eat(&mut self, kind: TokenKind) -> bool` -- if at(kind) then advance and return true, else false
- `expect(&mut self, kind: TokenKind) -> Result<Span, ()>` -- if at(kind) then advance and return Ok(span), else emit Syn050 diagnostic with message "expected {kind:?}, found {actual:?}" and return Err(())
- `expect_ident(&mut self) -> Result<(Spur, Span), ()>` -- if at_ident() then extract Spur from token, advance, return Ok, else emit Syn050 and Err
- `at_contextual(&self, name: &str) -> bool` -- if let TokenKind::Ident(spur) = self.peek() { self.interner.resolve(&spur) == name } else { false }
- `expect_contextual(&mut self, name: &str) -> Result<Span, ()>` -- if at_contextual(name) then advance and return Ok(span), else emit Syn050 with message "expected '{name}'" and Err

**Diagnostic emission:**
- `emit_error(&mut self, code: ErrorCode, message: String, span: Span, label: String)` -- pushes Diagnostic::new(code, message, span, Severity::Error, label) to self.diagnostics
- `emit_error_with_hint(&mut self, code: ErrorCode, message: String, span: Span, label: String, hint: String)` -- same but with .with_hint(hint)

**Error recovery:**
- `synchronize(&mut self)` -- skip tokens until reaching a declaration keyword (KwModel, KwSchema, KwPrompt, KwTool, KwAgent, KwImport, KwLet, KwPipeline, KwEnum, Eof) at brace depth 0, or a RBrace that brings brace depth to 0. Track brace depth: increment on LBrace, decrement on RBrace. Only consider keywords when brace_depth == 0.
- `error_limit_reached(&self) -> bool` -- count Error/Fatal severity diagnostics, return true if >= 20

**Source text access:**
- `span_text(&self, span: &Span) -> &str` -- `&self.source[span.clone()]`
